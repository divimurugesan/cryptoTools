# AI-Based Cryptanalysis Tool

A sophisticated Python-based cryptanalysis system that uses Artificial Intelligence and Machine Learning to break classical ciphers.

## Features

- 🔐 **Multiple Cipher Support**: Caesar, Vigenère, Monoalphabetic, and Playfair ciphers
- 🤖 **ML-Powered Analysis**: Random Forest classifiers for cipher type detection
- 📊 **Frequency Analysis**: Visual representation of letter frequencies
- 🎯 **N-Gram Scoring**: Advanced pattern recognition using n-gram analysis
- 📚 **Dictionary Attacks**: Word matching for improved accuracy
- 📈 **Confidence Scoring**: Quantitative measure of decryption success
- 🌐 **Web Interface**: User-friendly Flask web application
